Scalable Probabilistic Estimates of EV Charging (SPEECh)

Project

SPEECh is a probabilistic, data-driven model for large-scale scenarios of EV charging demand. The project is part of EV50, a flagship program under the Stanford Precourt Institute for Energy’s Bits & Watts Initiative. This web application was developed by Lesley Ryan and Siobhan Powell.

Publication:
Powell, S., Cezar, G. V., & Rajagopal, R. (2021). "Scalable Probabilistic Estimates of Electric Vehicle Charging Given Observed Driver Behavior." Under review.

Motivation

EV targets / growth

Long term planning to support electric vehicles (EVs) depends on detailed estimates of the future demand for charging. These estimates need to be: scalable, to enable real-time interaction with the model and represent the millions of EVs in upcoming targets; grounded in driver behaviour, which impacts every aspect of charging demand; and probabilistic, to capture the uncertainty in future assumptions and in drivers’ charging decisions.

About SPEECh

Our proposed model, SPEECh, addresses that intersection: a holistic, probabilistic model for large-scale scenarios of EV charging, grounded in data-driven modeling of real driver behavior. The model has three main components: driver groups, segment choices, and sessions behaviors.

Driver Groups

The driver groups divide drivers into 16 distinct behavior patterns. Redistributing the drivers by changing the distribution over the groups allows the modeler to reshape the overall charging profile. This can be adjusted within the web-app. For example, to simulate the result of policies supporting installation of charging stations in multi-unit dwellings, the modeler could increase the weight of group 10.

Segments

Each driver groups’ probability of choosing to charge in each segment on a given day is modeled. The segments are: workplace level 2 (L2), single family home residential L2, multi-unit dwelling L2, public L2, and public fast charging (DCFC).

Sessions Behaviors

For each driver group and each charging segment, e.g. for group 1 workplace charging, we model the charging behaviors using a mixture model. Each component in the mixture is a unique charging behavior. Changing the weights of the components allows the modeler to adjust the distribution over behaviors and reshape the charging segment. This can be adjusted within the web-app.

About the Data

For the results shown in the publication and on this web page, the model was trained on a large private data set of over 4 million charging sessions. The sessions took place in 2019 in the San Francisco Bay Area, California, and represent the charging of over thirty thousand individual drivers.

Code

The model is open-source; we have published code to help other researchers run SPEECh with their own data ([link]). This web-app runs off the same code. It runs the full model - each time the user changes an input, a new scenario is generated. This lets users interact with the model, customize scenarios, and understand the impact of their assumptions.

The model can rapidly generate new scenarios: on a laptop computer, a scenario of 8 million vehicles takes approximately 1 minute to run. In the web-app, the run time may depend on the number of users interacting with the tool at the same time.
